# 🚀 Email Automation & Reminder System

A full-stack Email Automation Platform built using Python, FastAPI, SMTP, SQLite, and Next.js.

## ✨ Features

- Bulk Email Sending
- CSV Contact Import
- Personalized Templates
- Async Email Processing
- FastAPI REST APIs
- SQLite Email History
- Next.js Dashboard
- SMTP Integration
- Delivery Tracking

---

## 🛠 Tech Stack

### Backend
- Python
- FastAPI
- AsyncIO
- SQLAlchemy
- SQLite
- SMTP

### Frontend
- Next.js
- React
- TailwindCSS
